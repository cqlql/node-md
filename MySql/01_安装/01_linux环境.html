                              以下是对 MySQL Community 版本进行的安装

                              # ubuntu 下安装

                              # apt 安装

                              先下载 MySQL APT Repository，最新下载地址打开此链接open in new window查看

                              sudo wget -P /home https://dev.mysql.com/get/mysql-apt-config_0.8.23-1_all.deb

                              解压

                              sudo dpkg -i mysql-apt-config_w.x.y-z_all.deb

                              更新包信息

                              sudo apt-get update

                              安装

                              sudo apt-get install -y mysql-server

                              参考文档：

                              MySQL 官方 :: A Quick Guide to Using the MySQL APT Repositoryopen in new window

                              # 查找 mysql 安装目录

                              当前环境是 window 子系统，所以排除 /mnt 目录

                              sudo find / -path /mnt -prune , -name mysql

                              # windows linux 子系统（wsl）安装 - 失败

                              失败原因

                              此安装方式在 wsl 中不能成功启动，目前没找到好的解决的办法，似乎与 wsl 没有 systemctl 有关。 wsl 环境安装方式见下

                              # 更新 Ubuntu 软件源头
sudo apt update

# 安装
sudo apt install -y mysql-server mysql-common

# 启动
sudo service mysql start

# 查看状态
sudo service mysql status

# 接下来进入 mysql 添加普通用户，再用普通用户登录
sudo mysql
