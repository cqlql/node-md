## css class 命名规范

```css
/* test-area 模块 */
.test-area {
}
/* test-area 模块子项 header */
.test-area_header {
}
/* 子项 header 子项 title*/
.test-area_header-title {
  /* 次要项 */
  .left {
  }
  .price {
  }
}
```
