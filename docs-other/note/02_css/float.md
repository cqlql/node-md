


## 浮动与非浮动，发生重叠

现有浮动元素 fA，非浮动元素 B。fA 在前，B 在后。

- B 将无视 fA 的存在，发生重叠。
- 并且 fA 层叠更高，将覆盖 B（相同 position 情况）

所以才需要清浮动

### 压缩报错问题

作为字面量对象的名称时，尽量以字符串形式。否则无法压缩。原因是float为 javascript 预留保留词

```js
var b = { float:'xx' }; // 不推荐
var b = { 'float': 'xx' }; // 推荐
```
