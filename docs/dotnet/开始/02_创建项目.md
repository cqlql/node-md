## dotnet cli

### webapi

#### 命令

```sh
# 1. 当前目录下创建
dotnet new webapi

# 2. TodoApi 目录下创建
# o 为 output 的缩写
dotnet new webapi -o TodoApi

# 项目根目录下运行
# 将会自动下载依赖
dotnet run

```

#### 初始目录介绍

obj 目录: 临时目录，不需要 git 保存

### console

```sh
dotnet new console --output sample1
```

### classlib 创建类库(dll)

```sh
dotnet new classlib
```

## visual studio

### web mvc

#### Enable Razor Runtime Compilation 选项

开启 Razor 文件 (.cshtml), 在构建发布时编译，能提供更好的性能。

如果开启运行时编辑，则能实时查看修改后的视图，提高开发效率。
