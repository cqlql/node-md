## 官方在线镜像仓库

[https://hub.docker.com/](https://hub.docker.com/)

## 镜像指令

### 查看 docker 中已有镜像

```sh

docker images
```

## 容器指令

### 查看所有容器

```sh
docker ps -a
```

### 只查看运行中的容器

```sh
docker ps
```

### 进入容器内部

```sh
docker exec -it 289cc00dc5ed bash
```
