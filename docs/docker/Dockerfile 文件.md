## 理解

通过脚本创建自定义 docker 镜像



>
> **from 指令**
>
> 减少镜像体积，指定某一阶段开始构建镜像。比如最终镜像只需要运行环境，但其中因为需要还有编译环境，这就大大增加了镜像的体积。
